# 08 BRAINSTORM IDEAS

## Feature Ideas
- **Infinite Mode:** Cards keep appearing as long as you don't make a mistake.
- **Themed Decks:** Allow users to choose between numbers, emojis, or colors.
- **Sound Effects:** Add subtle audio feedback for flipping, success, and errors.
- **Global Leaderboard:** Integrate a simple backend (or local storage) to track high scores.

## Technical Ideas
- **Web Component Migration:** Refactor cards into native Web Components.
- **Tailwind Integration:** Switch from Vanilla CSS to Tailwind for faster styling iteration.
- **PWA Support:** Make the game playable offline with a Service Worker.
