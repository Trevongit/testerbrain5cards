# 06 INTELLIGENT DEV METHOD

## Standards
- Clean, documented Python code.
- Surgical edits only (minimal diffs).
- Maintain portability (Linux/macOS/Windows where practical).

## Workflow
1.  **Research:** Map existing patterns.
2.  **Strategy:** Propose targeted change.
3.  **Execute:** Implement and verify.
4.  **Log:** Record major merges in `MERGE_LOG.md`.
